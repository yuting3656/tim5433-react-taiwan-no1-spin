# Installing

```bash

```

# Basic Example

```js
import "./App.css";
import { TaiwanNo1 } from "react-taiwan-no1-spin";

function App() {
  return (
    <div className="App">
      <TaiwanNo1 width="390" speed="1" />
    </div>
  );
}

export default App;
```
